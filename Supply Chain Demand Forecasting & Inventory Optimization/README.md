## âš™ï¸ Setup & Usage

### Python Notebooks
```bash
pip install -r requirements.txt
jupyter notebook notebooks/01_data_prep_eda.ipynb
```

### Run Forecasting Pipeline Directly
```bash
cd src
python demand_forecasting.py      # Outputs forecast CSVs + charts
python inventory_optimization.py  # Outputs inventory CSV + charts
```

### Excel Model
1. Open `excel/SupplyChain_Forecast_Inventory.xlsx`
2. Update **blue cells** in `Inventory_Optimization` sheet (current stock, lead time, etc.)
3. All downstream calculations auto-update
4. `Stock_Health` sheet shows live alert status

### Power BI Dashboard
1. Open `powerbi/SupplyChain_Dashboard_PLACEHOLDER.txt` for rebuild instructions
2. Import processed CSVs from `data/processed/`
3. Apply all DAX measures from `powerbi/dax_measures.md`
